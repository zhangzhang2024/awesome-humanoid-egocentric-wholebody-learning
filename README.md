# Awesome-Egocentric-Humanoid-Learning [![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome)

This repo collects benchmarks, datasets, papers, projects and codes about egocentric wholebody learning for humanoid robots.

If you find this repository useful, please consider  **giving us a star** 🌟
